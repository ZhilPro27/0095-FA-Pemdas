Sistem Manajemen Kelulusan MataKuliah

Pengantar

Selamat datang di dokumentasi Sistem Manajemen Kelulusan MataKuliah. Sistem ini dirancang untuk mengecek kelulusan MataKuliah berdasarkan nilai yang sudah di-input.

Struktur Kode

Kode ini terdiri dari tiga kelas utama:

  • MataKuliah : Kelas dasar yang memiliki function-function yang akan diturunkan ke kelas turunannya.
  • Pemrograman : Kelas turunan dari kelas MataKuliah yang meng-override beberapa function parent-nya.
  • Jaringan : Kelas turunan dari kelas MataKUliah yang meng-override beberapa function parent-nya.

Setup dan Konfigurasi

Untuk menjalankan sistem ini, pastikan anda memiliki compiler yang mendukung standar operasi. Kompilasi dan jalankan file ini menggunakan compiler pilihan anda.

Tentang Pengembang
